# Cyber Triage Tool for LLM Security

## Overview
This project implements a forensic triage tool for detecting and investigating
prompt injection and data exfiltration attempts in LLM-integrated systems.

## Features
- Flask API for ingesting LLM logs
- Heuristic-based threat scoring
- JSON-based forensic logging
- Investigator dashboard (Tailwind UI)

## Requirements
- Python 3.9+
- Flask

## Setup
```bash
pip install flask
python app.py
